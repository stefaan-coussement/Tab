### tab.doReturn( ?value, ...extraValues ) Â» tab

````
tab.doReturn(value) ~ tab.return(value)
````

parameters:
*   **value** : *any*
    the principal value to store in this tab.

*   **...extraValues** : *...any*
    extra information to store in this tab.

returns:
*   **tab** : *[object Tab]*
    this tab, updated with the given values.
